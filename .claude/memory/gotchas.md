# Gotchas Memory

Use this file to record non-obvious pitfalls that repeatedly matter in this repository.

## What belongs here

- Hidden constraints that are easy to miss during normal code reading
- Workflow traps that have caused or are likely to cause mistakes
- Important warnings that do not fit cleanly in architecture or conventions

## What does not belong here

- Generic advice
- Code facts that are already obvious from surrounding files
- Temporary warnings tied only to one in-progress task

## Entry format

### YYYY-MM-DD — Gotcha title
- **Gotcha:** The pitfall or warning
- **Why:** Why it matters
- **How to apply:** How future changes should avoid it
