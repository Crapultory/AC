# Bugfix Memory

Use this file to record durable lessons from real bug fixes.

## What belongs here

- Non-obvious root causes that are likely to recur
- Fix lessons that future sessions should remember before changing similar code
- Regressions that established a lasting guardrail

## What does not belong here

- One-off task notes
- Full debugging transcripts
- Repo structure summaries
- Facts that are easy to read directly from code or tests

## Entry format

### YYYY-MM-DD — Short lesson title
- **Lesson:** What future work should remember
- **Why:** Why this matters
- **How to apply:** Where this should influence future changes
